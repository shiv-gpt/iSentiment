Interactive SENTIMENT Analysis of Twitter Tweets

Welcome to iSentiment, a basic webApp developed for Interactive Sentiment Analysis of Twitter Tweets.



Input a topic on which you want to perform Sentiment Analysis
Input number of tweets with which you want to perform Sentiment Analysis


Heads up! Due to Tweepy api restrictions, a maximum of 1000 tweets can be fetched at one time from the past 7 days. For Sentiment Analysis, TextBlob package was used.
